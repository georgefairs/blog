This is a test of embedding a [marimo](https://marimo.io) notebook directly in a post.
marimo notebooks export to **WebAssembly**, so the Python below runs entirely in your
browser — there's no server, which means it works fine on a static site like this one.

The example is the **logistic map**, `xₙ₊₁ = r·xₙ(1 − xₙ)` — a one-line model of
constrained growth that slides from a stable orbit into full chaos as you push the growth
rate `r` past ~3.57. Drag the sliders:

It can take a few seconds on first load while the WebAssembly runtime and Python packages
(`numpy`, `matplotlib`) download and start up.

---

**How this was made:** the notebook lives at `notebooks/logistic_map.py` in this site's
repo. The `<iframe>` above points at [marimo.app](https://marimo.app), which fetches that
file from GitHub and runs it in your browser via WebAssembly — so nothing but the small
notebook file is committed here.
